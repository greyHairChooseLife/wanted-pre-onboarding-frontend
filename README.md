# 원티드 프리온보딩 프론트엔드 - 선발 과제

선발 과제 제출용 프로젝트입니다.
<br/>
<br/>

### 프로젝트 실행환경

node version 16.15.0 이상을 권장합니다.
<br/>
<br/>

### 프로젝트 설치 및 실행방법

1. 저장소를 `git clone`합니다.
2. 설치된 폴더에서 의존성 패키지를 설치(`npm install`) 해 줍니다.
3. 스크립트를 실행(`npm run start`)하여 프로그램을 실행합니다.
